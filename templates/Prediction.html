Fetal Health Prediction Model

Cardiotocography (CTG) is a critical tool for monitoring fetal heart rate and uterine contractions during pregnancy and labor. It helps clinicians detect potential issues early, such as oxygen deprivation, which is vital for ensuring the well-being of both mother and baby. This project focuses on using CTG data to predict fetal health status by classifying it into three categories: Normal, Suspect, and Pathological. Early identification of these conditions can guide timely medical interventions.

To address this, a machine learning model was developed to analyze key CTG features and classify fetal health. Using supervised learning, the model accurately identifies patterns and categorizes fetal health. It was evaluated through metrics such as accuracy and precision, showing reliable performance, especially in identifying pathological cases. The model provides a proactive approach to improving healthcare decisions by offering timely insights into fetal health conditions.
